# gd-zy-wechat

基于`node v12.16.1` `npm v6.13.4`开发环境搭建的`Vue2.x + Vant2 + Less + Vue Router + Vuex`项目

## 脚本运行

```bash
yarn install        # 依赖安装
yarn serve          # 本地运行
yarn dev            # 本地运行
yarn build          # 正式环境打包
yarn build:test     # 测试环境打包
yarn lint           # 格式化代码
```

### 文件命名

- **\*.vue** 在 views 目录下 命名规范同 js 命名规范；组件文件则以驼峰命名且首字母大写(如 MyCom)
- **\*.js** 命名遵循一个单词以全小写 name，多单词则以`-`连接 kebab-case
